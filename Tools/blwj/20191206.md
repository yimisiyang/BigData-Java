# 一、基础知识 #

## # 1.虚拟机安装 # ##

分区三个目录：/boot、 /、 /swap。

## 2.查看配置网络IP ##

方式1：

虚拟机菜单栏 编辑-->虚拟网络编辑器

方式2：

打开windows系统网络配置进行设置

## 3.CentOS网络设置 ##

1.设置成静态IP （通过桌面方式）
 
`ifconfig` 查看网络ip.
 
`server network restart`  重启网络配置

2.通过命令行模式

`vi /etc/sysconfig/network-scripts/ifcfg-eth0`


**遇到问题：**  

`./network-scripts` 下没有`ifcfg-eth0` 出现的是 `ifcfg-ens33`

**解决办法：**

https://blog.csdn.net/TymonPaul/article/details/80188177
